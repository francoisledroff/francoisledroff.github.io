First published on August 08 2018 and tagged Adobe . IMS . adobe.io . Security . Java . OAuth . JWT . API .

I just published on github a small Java utility showcasing Adobe IMS JWT Token exchange.

It might help you understanding and implementing your own integration against Adobe APIs (that requires the use of JWT token exchange)
